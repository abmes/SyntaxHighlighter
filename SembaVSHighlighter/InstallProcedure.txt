1. Ensure no previous version of SembaVSHighlighter is instlled

2. Add Extension Gallery if not already added
  - Start Visual Studio
  - go to Tools -> Options -> Environment -> Extensions and Updates
  - add Extension Gallery
    - Name: "Semba Extensions" 
    - URL: "http://semba.bg/vstools/atom.xml"

3. Go to Tools -> Extensions and Updates -> Online -> Semba Extensions and install the extension
