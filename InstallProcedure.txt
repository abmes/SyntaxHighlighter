1. Ensure no previous version of Abmes.SyntaxHighlighter is instlled

2. Add Extension Gallery if not already added
  - Start Visual Studio
  - go to "Tools -> Options -> Environment -> Extensions and Updates"
  - add Extension Gallery
    - Name: "Abmes Extensions" 
    - URL: "http:///dev/vstools/atom.xml"

3. Go to "Tools -> Extensions and Updates -> Online -> Abmes Extensions" and install the extension

4. Go to "Tools -> Extensions and Updates -> Installed -> Abmes.SyntaxHighlighter -> More Information" to view the colors reference

5. Go to "Tools -> Options -> Environment -> Fonts and Colors" to set the colors
